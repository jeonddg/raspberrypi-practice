# Raspberry Pi Practice

라즈베리파이를 활용한 GPIO 제어 및 IoT 실습을 정리한 저장소입니다.  
다양한 하드웨어 제어 실습을 통해 임베디드 시스템의 동작 원리와 구조를 학습합니다.

---

## Overview

- GPIO 기반 하드웨어 제어 실습
- 센서 및 액추에이터 인터페이스 구현
- IoT 시스템 기초 구조 이해

---

## Development Environment

- **Board** : Raspberry Pi 5  
- **OS** : Raspberry Pi OS  
- **Language** : Python 3  
- **Library** : gpiozero
