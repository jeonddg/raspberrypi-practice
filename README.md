# Raspberry Pi Practice

라즈베리파이를 이용한 **GPIO 제어 및 IoT 실습 코드**를 정리하는 저장소입니다.
LED 제어, 센서 인터페이스, 통신 모듈 등을 실습하며 라즈베리파이 기반 **하드웨어 제어와 IoT 시스템 구조**를 학습합니다.

---

## Development Environment

* **Board** : Raspberry Pi 5
* **OS** : Raspberry Pi OS
* **Language** : Python 3
* **Library** : gpiozero
